O(1), by construction

A hash map and a linked list, doing one job each.

The map answers “where is this key?” in constant time. The list answers “what have we touched least recently?” — also in constant time. Neither ever walks the collection.

"flag"→ListNode #1

recency order — a circular doubly-linked list
#1 head⇄ #3⇄ #2 tail

Lookup goes through the map: key → node, in O(1).

Touch a node on GET and it's spliced out and re-linked at the head — a handful of pointer rewires, independent of how many entries the cache holds.

Evict when full: the tail is the least recently used entry by definition, so removing it is the same O(1) splice. Expired nodes are reclaimed first.

Decisions worth defending

The interesting choices aren't the data structures.

They're the edge cases — the ones that separate a toy from something you'd trust with a process's memory.

Recency

A circular doubly-linked list

Closing the ring makes the head and tail neighbours, so promoting an entry and evicting the least-used one are the same splice logic — no special-casing the ends, and no head/tail sentinel bookkeeping.

Expiration

Lazy TTL, with a bounded sweep

A key is gone the instant its deadline passes and is dropped on the next access — no background timer per key. A capacity-triggered sweep reclaims expired nodes before evicting a live one, so dead entries can't crowd out good ones.

Durability

Snapshots a reader can't tear

SAVE serializes live entries to a temp file, fsyncs it, then atomically renames it into place — so a concurrent reader never observes a half-written file. TTLs are stored as absolute deadlines, so expiry survives a restart.

Lifecycle

Graceful shutdown

SIGINT/SIGTERM stops accepting connections and flushes a final snapshot before exit. A guard ensures a second Ctrl-C won't kick off a concurrent save and corrupt the write.

The wire protocol

Newline-terminated, human-typable.

Commands are case-insensitive; a value is the remainder of the line, so it may contain spaces. Everything in the console above speaks exactly this protocol.
